# 文档模板

该目录包含状态化任务文档体系下的可复用模板。

## 模板列表
- [active-task-template.md](./active-task-template.md) - 新建 `docs/active-work/<task>/` 任务时使用
- [roadmap-item-template.md](./roadmap-item-template.md) - 新建 `docs/roadmap/` 延期事项时使用
- [completed-task-template.md](./completed-task-template.md) - 归档到 `docs/completed/` 时使用

## 使用规则
必须从与任务状态匹配的模板开始。
不要让一份文档同时承担 active、roadmap 和 completed 的混合语义。
